* Ezt a könyvtárat majd nevezzék át
  a saját nevükre, és írják be a Neptun-kódjukat is!

  Az egyes részeket aláhúzásjellel válasszák el!
  A könyvtár nevében NE használjanak ékezeteket!

* Mit lehet használni?

  - saját írott jegyzet
  - hivatalos C# API dokumentáció

  Minden egyéb eszköz használata meg nem engedett segédeszköznek minősül.
  Korábban megírt forráskódok (állományok) sem használhatók!

* Meg nem engedett segédeszköz használata esetén a dolgozatra
  0 pontot kapnak.

* Az egyes feladatokat a megfelelő alkönyvtárban dolgozzák ki!

* A feladatokat az eddig tanult módszerekkel meg lehet oldani!
  Olyan megoldásokat várok, amikben az eddig tanultak
  köszönnek vissza!

* Ha valamelyik feladatnál fájlból kell olvasni, akkor az
  input állomány mellékelve lesz a feladat mappájában.

  Az input állományokat tilos módosítani!

* Minden egyes feladat megoldása kész, komplett, minden mástól
  független megoldás legyen. Vagyis a projekt a feladat mappájába
  belépve egyszerűen fordítható és futtatható legyen.

  Példa:

      $ cd feladatN
      $ dotnet run         # lefordul és lefut

* Minden feladatot külön mappában dolgozzon ki. Beküldéskor a `bin/` és `obj/` mappák
  kivételével minden file-t küldjön el (a `.csproj` állományt is).
  Vagyis ha kibontom a ZIP-et s belépek egy projektmappába, akkor a projektet a
  "dotnet run" parancs kiadásával le tudjam fordítani / futtatni.

* Ha kész vannak, akkor majd ezt a könyvtárat kell
  becsomagolni ZIP-be és feltölteni a tárgy Moodle oldalára.

  Példa:

      $ zip -r zh.zip Pelda_Peter_ABCDEF/
